# Getting Started with Create React App

This project is used as a presentation for the Culture Trip's Web Guild to introduce React to the engineering team.

It contains some concepts about atoms and selectors, showing how data can be fetched by either using effects or by using async selectors.

## Available Scripts

In the project directory, you can run:

### `npm run start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
